Usage guide and feature
1. First you need to register candidates using API http://localhost:8080/entercandidate?name=
2. After registration completed your able to cast vote to your candidate use API http://localhost:8080/castvote?name=
3. Get vote count of your candidate use API http://localhost:8080/countvote?name=ajay
4. You can get List of candidate with vote count use API http://localhost:8080/listvote
5. get winner use API http://localhost:8080/getwinner
